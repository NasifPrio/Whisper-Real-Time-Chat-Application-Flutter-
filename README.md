# 💬 Whisper - Real-Time Chat Application (Flutter)

## 📌 Project Overview

A real-time chat application built with Flutter that supports user authentication, instant messaging, and profile updates. The app is designed to provide a smooth and responsive chatting experience.

---

## 🎯 Purpose

To enable simple, fast, and secure communication for users such as students and small communities.

---

## 🚀 Main Goal

Build a lightweight, user-friendly chat platform with real-time capabilities and secure messaging.

---

## 🛠️ Tech Stack

* **Frontend & UI:** Flutter
* **Language:** Dart
* **Authentication:** Firebase Authentication
* **Database:** Firebase Realtime Database
* **Image Storage:** Cloudinary
* **Encryption:** AES (CBC Mode)

---

## ✨ Features

* 🔐 User Sign-Up, Login & Logout
* 💬 Real-Time Messaging
* 🖼️ Image Sharing (via Cloudinary)
* 👤 Profile Update with Profile Picture
* 🔒 End-to-End Message Encryption (AES)
* 📡 Live UI Updates using StreamBuilder
* ⚡ Fast and Responsive UI

---

## 🏆 Accomplishments

* Implemented secure user authentication using Firebase
* Enabled seamless real-time messaging with Firebase Realtime Database
* Integrated Cloudinary for efficient image upload and retrieval
* Built a clean and responsive UI using Flutter
* Managed navigation and dynamic state updates across multiple screens
* Applied AES encryption (CBC mode) for secure message transmission

---

## 🔬 Technical Insights

* Firebase Realtime Database ensures smooth and instant data synchronization
* Cloudinary provides fast and reliable cloud-based image handling
* StreamBuilder simplifies real-time UI updates in Flutter applications
* AES encryption enhances message security during transmission
